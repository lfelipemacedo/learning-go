# Tipos de Dados Simples

## Objetivo
Revisar os tipos básicos e suas peculiaridades no Go, com exemplos práticos.

## Conteúdo
- **Strings:** interpretadas vs. raw strings.
- **Números:** `int`, `uint`, `float32`, `float64` e `complex`.
- **Booleanos:** valores `true/false` e zero value.
- **Errors:** `error` como valor e `nil` como ausência de erro.
- **Variáveis e constantes:** tipagem explícita e inferida.

## Exemplo no código
O arquivo `simple-data-types.go` imprime exemplos de cada item acima.

## Como executar
```bash
go run .
```
