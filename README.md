# 🚀 Estudos em Go (Golang)

Repositório com meus estudos de Go. As pastas numeradas agrupam exemplos práticos e notas de cada tema.

## Estrutura
- `1-first-examples/` — primeiros exemplos (Hello World, CLI, web server).
- `2-simple-data-types/` — tipos simples (strings, números, booleanos, erros, variáveis e constantes).
- `3-pointers-and-values/` — valores e ponteiros.
- `4-aggregate-data-types/` — arrays, slices, maps e structs.

## Como executar
Cada pasta tem seu próprio `go.mod`. Entre na pasta desejada e rode:
```bash
go run .
```
