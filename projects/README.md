# Projetos Práticos

## Objetivo
Esta pasta reúne os projetos da trilha prática que consolidam os estudos do repositório. A proposta é sair dos exemplos isolados e aplicar os conceitos em implementações mais completas, do mais básico ao mais avançado.

## Ordem sugerida
1. `01-cli-tarefas/`
2. `02-parser-library/`
3. `03-discount-engine/`
4. `04-product-catalog-api/`
5. `05-order-system/`
6. `06-file-processing-pipeline/`
7. `07-payment-integration/`
8. `08-portfolio-project/`

## Como estudar
- avance em sequência, reaproveitando o que foi aprendido nas etapas anteriores;
- trate cada projeto como uma entrega incremental, começando pelo caminho feliz;
- adicione testes, documentação e refinamentos conforme o projeto evolui;
- revise critérios de aceite e próximos passos antes de considerar cada projeto concluído.
